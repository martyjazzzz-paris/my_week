# Моя неделя

Личный недельный планировщик. React + Vite, один компонент (`src/App.jsx`), без бэкенда — все данные хранятся локально на устройстве (`localStorage`).

## Деплой на Vercel

1. Залить эту папку в GitHub-репозиторий (веб-интерфейс: Add file → Upload files, либо `git push`).
2. На vercel.com → New Project → выбрать репозиторий → Deploy.
   Vercel сам определит Vite и запустит `npm run build` — ничего настраивать не нужно.
3. Открыть полученную ссылку (`*.vercel.app`) в Safari на iPhone.
4. Поделиться → «На экран Домой» — появится иконка, приложение откроется на весь экран без адресной строки.

## Что не работает в превью артефакта, но заработает здесь
- Голосовой ввод (микрофон)
- Уведомления за N минут до дела (пока вкладка открыта на переднем плане — фоновые пуши это отдельная, более сложная задача)
- PIN-экран как реальная точка входа

## Локальная разработка
```
npm install
npm run dev
```

## Структура
- `src/App.jsx` — всё приложение целиком (по договорённости — один файл)
- `public/manifest.webmanifest`, `public/sw.js` — PWA
- `public/icon-*.png` — иконки для экрана Домой
